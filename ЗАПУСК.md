# 🚀 Запуск проекта ИСТОК

## 📋 Требования

- **Go** 1.24.9+
- **Node.js** 18+
- **npm** или **bun**
- **Git**

## ⚙️ Настройка переменных окружения

### Backend (.env в корне проекта)

```bash
OPENROUTER_API_KEY=your_openrouter_api_key_here
PORT=8080
```

### Frontend (web/.env.local)

```bash
NEXT_PUBLIC_API_URL=http://localhost:8080
```

## 🐛 Решение проблем

### Backend не запускается

```bash
# Проверьте переменные окружения
echo $env:OPENROUTER_API_KEY  # Windows
echo $OPENROUTER_API_KEY      # Linux/Mac

# Проверьте порт
netstat -ano | findstr :8080  # Windows
lsof -i :8080                 # Linux/Mac
```

### Frontend показывает "Агент подключается к мозгу..."

1. Убедитесь, что backend запущен на порту 8080
2. Проверьте `web/.env.local`:
   ```
   NEXT_PUBLIC_API_URL=http://localhost:8080
   ```
3. Перезапустите frontend

### Ошибка CORS

Backend уже настроен на CORS. Если проблема сохраняется:
- Проверьте, что frontend на `localhost:3000`
- Очистите кэш браузера
